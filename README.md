
# VentasFix

Sistema de gestión de ventas desarrollado en Laravel. Permite la administración de usuarios, productos y clientes, así como la visualización de estadísticas en un dashboard web.

## Funcionalidades

### 1. Usuarios (WEB y API)
1.1. Listar todos los usuarios.
1.2. Obtener los datos de un usuario por su ID.
1.3. Agregar un nuevo usuario.
1.4. Actualizar un usuario por su id.
1.5. Eliminar un usuario por su Id.

### 2. Productos (WEB y API)
2.1. Listar todos los productos.
2.2. Obtener los datos de un producto por su ID.
2.3. Agregar un nuevo producto.
2.4. Actualizar un producto por su id.
2.5. Eliminar un producto por su Id.

### 3. Clientes (WEB y API)
3.1. Listar todos los clientes.
3.2. Obtener los datos de un cliente por su ID.
3.3. Agregar un nuevo cliente.
3.4. Actualizar un cliente por su id.
3.5. Eliminar un cliente por su Id.

### 4. Dashboard (SOLO WEB)
4.1. Información de cuántos usuarios tiene el sistema.
4.2. Información de cuántos productos tiene el sistema.
4.3. Información de cuántos clientes tiene el sistema.


## Template utilizado

El sistema utiliza el template [Vuexy - Pixinvent](https://demos.pixinvent.com/vuexy-html-admin-template/documentation/quick-start.html) para la interfaz de usuario web, junto con Tailwind CSS y Vite para la gestión de assets (este fue entregado por la institucion).
